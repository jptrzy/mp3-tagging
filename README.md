Ogg-tagging is my personal script to automatically tag ~~mp3~~ ogg.

## Installation

### Manual/Git install
```
git clone https://github.com/jptrzy/ogg-tagging
cd ogg-tagging
make
sudo make install
```

### UnInstallation
```
sudo make uninstall
```
