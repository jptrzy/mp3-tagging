# mp3-tagging
MP3-tagging is my personal script to automatically tag mp3.

## Installation

### Manual/Git install
```
git clone https://github.com/jptrzy/mp3-tagging
cd mp3-tagging
sudo python setup.py install
```

### UnInstallation
```
sudo pip uninstall mp3-tagging
```
